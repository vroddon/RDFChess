Linked Chess Data

This site provides the models for representing chess games and a dataset of games in RDF, as well as a service to transform PGN games into RDF game

Dataset

Download the original collection in PGN with 2,883,907 games. pgn.tar (2 Gb compressed)

PGN dataset

Download a single RDF dump with 4885 games (whiteplayer's name starting with 'Q') Q.nt.zip (20 Mb, uncompressed 210 Mb)

RDF sample

Download the complete RDF games collection with 2,883,907 games. rdfchess.tar (12 Gb compressed)
